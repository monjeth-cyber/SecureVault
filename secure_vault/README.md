# 🔐 SecureVault - Flutter Secure Authentication App

A production-ready Flutter authentication app with **state-of-the-art security** and **strict MVVM architecture**.

## 🚀 Quick Start (45 minutes to working app!)

### Step 1: Configure Firebase (15 min)
1. Open `FIREBASE_QUICKSTART.md` in this folder
2. Download `google-services.json` from Firebase Console
3. Place at: `android/app/google-services.json`
4. Add SHA-1 fingerprint to Firebase

### Step 2: Run the App (1 min)
```bash
flutter clean
flutter pub get
flutter run
```

### Step 3: Test All Features (30 min)
- Register with email → verify email → login
- Try Google Sign-In
- Try Facebook Login
- Enable biometric authentication
